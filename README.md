# SynAI

Modern AI Chat Interface

## Overview
SynAI is a clean, responsive web application designed for AI conversations with multiple providers.

## Features
- Multi-provider AI integration
- Real-time chat interface
- Modern React architecture
- TypeScript for type safety
- Responsive design

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

## Development
```bash
npm run build    # Production build
npm run test     # Run tests
npm run lint     # Code linting
```
